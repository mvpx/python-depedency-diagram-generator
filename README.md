# Diagrams Project

A Python project for creating diagrams.

## Setup

1. Activate the virtual environment:
   ```
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```
   uv pip install -e .
   ```

3. Run tests:
   ```
   python -m pytest
   ```

## Project Structure

- `src/diagrams/`: Main package code
- `tests/`: Test files
- `pyproject.toml`: Project configuration
